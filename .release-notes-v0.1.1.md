## v0.1.1 (2026-07-20)

Security hardening release for the OData and SyncFeed clients.

### Security

- **Safe XML parsing** for SyncFeed: external entities and network access disabled (`resolve_entities=False`, `no_network=True`)
- **Same-origin checks** for OData `@odata.nextLink` and SyncFeed pagination URLs (blocks SSRF-style host jumps)
- **UUID-only entity IDs** on get/download paths (rejects path-injection style values)
- New exception: `SecurityError`

### Install

```bash
pip install -U tweedekamer_opendata
```
